# Word Definer    

#### By Bryce Bresnan <Bryce.Bresnan@gmail.com>

#### A project to demonstrate routing in Sinatra with a children's website that allows kids to write up a list of words that they don't know (but would like to learn) and then come up with their own definitions. 

## Technologies Used

* Ruby
* Sinatra

## Setup/Installation Requirements

* Clone this repository from GitHub.
* Navigate to the root in command line
* Run <code>$bundle install</code> to install the required gems.
* Run <code>$ruby app.rb</code> to start the server.
* Navigate in your browser to http://localhost:4567/
